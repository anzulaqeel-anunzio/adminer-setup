# Adminer Docker Setup

A Docker Compose configuration for Adminer, a lightweight database management tool (alternative to phpMyAdmin).

## Features

-   **Multi-DB Support**: MySQL, PostgreSQL, SQLite, MS SQL, Oracle.
-   **Lightweight**: Single PHP file based (containerized).
-   **Design**: Modern default theme.

## Installation

1.  **Clone the repository**.
2.  **Start Services**:
    ```bash
    docker-compose up -d
    ```
3.  Visit `http://localhost:8080`.

## Configuration

This setup includes a MySQL database for demonstration, but Adminer can connect to any database accessible from the container.
